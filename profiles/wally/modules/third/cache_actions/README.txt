Cache actions provides rules actions for clearing drupal caches.

It currently supports clearing:

* Drupal cache bins
* CSS/JS cache
* The cache of specific views
* The cache of specific panel page variants
* The cache of specific mini panel variants
* The cache of specific mini panel panes
* The cache of specific panel page panes
* The varnish cache (with the varnish module)

More extensive documentation can be found here:
http://drupal.org/node/927276
